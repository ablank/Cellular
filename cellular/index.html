Cellular is a responsive base theme for Drupal 7, designed to provide a more conventional workflow for front-end developers.

Theme settings (/admin/appearance/settings/cellular) provide a clean way to control the code generated by drupal and implement a wide range of features such as updating jQuery, providing social media links, and a lot of other cool stuff.

Features

Fine-tune Control of Resources

Tired of fighting styles set by modules? Get rid of that nonsense with cellular_remove_css() - Add the module/stylesheet to the array $exclude in /inc/css_alter.inc

Load css in the order you need with cellular_add_css() - Add your stylesheets to the array $add_css in /inc/css_alter.inc

Load scripts in the order you need with cellular_add_js() - Add your scripts to the array $add_js in /inc/js_alter.inc

jQuery Update

Update jQuery to the version of your choice (1.8.2, 1.7.1, 1.5.1, 1.4.4) without breaking Views from the Google or MS CDN with a local fallback

jQuery UI Update

Update jQueryUI to the version of your choice (1.10.1, 1.9.2, 1.8.11, 1.8.7)

Select the UI theme of your choice, or use the CUSTOM flat UI style for a lightweight theme that matches your site.

Full Menu Tree

Print all main menu items (not just parents) and style dropdowns with the fancy css/javascript you want.

SVG/PNG Icons

Favicons & apple-touch-icons

SVG icons in supported browsers, or falls back to PNG for older browsers.

Social media

Easily add links for users to follow or share your site with social media by selecting the desired links in theme settings.

Compass/SASS Styling

cd path/to/theme

compass watch

Mobile-first styling separated into logical partials for easy inclusion/exclusion of styles.

Grunt

Build & test your theme with ease

cd path/to/theme

npm install

grunt watch || dev || prod

Included/Optional Javascript Libraries

Cellular UI

UI elements (accordions, tabs, mobile menu, etc.) built for easy integration with Views.

Modernizr

Test browser feature support and load resources based on capabilities.

D3js

D3.js is a JavaScript library for manipulating documents based on data. D3 helps you bring data to life using HTML, SVG and CSS.

GSAP (GreenSock Animation Platform)

GSAP is a suite of tools for scripted, high-performance HTML5 animations that work in all major browsers.

THREEjs

A JavaScript 3D Library which makes WebGL simpler.

Masonry

Cascading grid layouts

Prism Syntax Hilighter

Code syntax hilighter.
