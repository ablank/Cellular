Cellular UI Javascript Components

jAccordion

  • Edit
  • Delete

Converts a View formatted as an HTML list with the class "jAccordion" into an accordion style display.

  • Edit view

  • Markup

    <ul class="jAccordion"> <li> <p>Title 1</p> <div> <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p> </div> </li> <li> <p>Title 2</p> <p>Pellentesque egestas mi nec odio convallis dapibus.</p> </li> <li> <p>Title 3</p> <p> Nullam eget arcu vulputate, tincidunt dolor quis, bibendum ex.</p> </li> </ul>

  • JS

    <script> $('.selector').jAccordion({ "active": 0, // Index value of initial content to display. "duration": 500, // Duration of transition. "easing": "swing", // Type of easing. "single": false // Allow multiple panels to be opened or only 1? }); </script>

  • CSS

jBlocklink

  • Edit
  • Delete

Wrap a block of content with a link.

  • Edit view

  • Markup

    <ul class="jScrolli"> <li> <p>Title 1</p> <div> <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p> </div> </li> <li> <p>Title 2</p> <p>Pellentesque egestas mi nec odio convallis dapibus.</p> </li> <li> <p>Title 3</p> <p> Nullam eget arcu vulputate, tincidunt dolor quis, bibendum ex.</p> </li> </ul>

  • JS

    <script> $('.selector').jBlocklink({ "cclass": "jBlocklink-link" // Class to add to the link for styling. }); </script>

  • CSS

jMmenu

  • Edit
  • Delete

Mobile friendly menu, adds menu button if window size is less than the set breakpoint.

  • Edit view

  • Markup

    <ul class="jMmenu"> <li><a href="#">Link</a></li> <li><a href="#">Link</a></li> <li><a href="#">Link</a></li> </ul>

  • JS

    <script> $('.selector').jMmenu({ "breakpoint": 650 // Window breakpoint trigger, in pixels }); </script>

  • CSS

jScrolli

  • Edit
  • Delete

Rotate display through a list of content.

  • Edit view

  • Markup

    <ul class="jScrolli"> <li> <p>Title 1</p> <div> <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p> </div> </li> <li> <p>Pellentesque egestas mi nec odio convallis dapibus.</p> </li> <li> <p> Nullam eget arcu vulputate, tincidunt dolor quis, bibendum ex.</p> </li> </ul>

  • JS

    <script> $('.selector').jScrolli({ "speed": 500, // Duration of cycle "pause": 3000 // Time to pause between cycles }); </script>
